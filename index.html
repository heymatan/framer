Device Frames

A set of reusable HTML/CSS frames for context of webpages, with a fallback when no page is displayed.

    Tenets:
  • All device sizes are declared in px. This is to ensure that the iframe sizing is correct even when zoomed (ems would resize the container).
  • The outermost container for a frame is a <div class="device-frame DEVICE-NAME"> where DEVICE-NAME is a unique identifier.
  • All content should go into a nested <div class="viewport">
  • More to be added...
